Логика получения данных

Получили перечень купонов со страницы одного бизнеса(Первый гет запрос)
и начинаем работать с каждыым из купонов отдельно


1: С основной страницы, из получаем json

2: Далее name и description получаем из json-а.
Также из json-а получаем offerId

3: С помощью offerId делаем запрос "https://www.groupon.com/coupons/redemption/ + offer_id"
и получаем второй json а в нем code

4: Во втором json-е по запросу "https://www.groupon.com/coupons/redemption/ + offer_id"
также берем merchantId и делаем Третий запрос "https://www.groupon.com/coupons/redirect/merchant/ + merchantId"
из которого получаем наш special_url
